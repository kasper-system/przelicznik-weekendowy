# JK. Kasper - Analiza grafikow

Jednoplikowa aplikacja webowa do analizy godzin weekendowych z grafikow Excel.

## Publikacja przez GitHub Pages

1. Utworz nowe repozytorium na GitHub.
2. Wgraj pliki:
   - `index.html`
   - `README.md`
   - `.gitignore`
3. Wejdz w `Settings` -> `Pages`.
4. Ustaw:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - folder: `/root`
5. Po chwili GitHub pokaze link do strony.

## Prywatnosc danych

Pliki Excel z grafikami nie sa wysylane na serwer aplikacji. Analiza odbywa sie lokalnie w przegladarce uzytkownika.

Nie wrzucaj plikow `.xlsx` z grafikami do publicznego repozytorium.
